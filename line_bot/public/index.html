LanguageBot🤖

AI 驅動的智能語言學習機器人

整合 OpenAI、語音辨識、TTS 等先進技術，為您打造全方位的英文學習體驗

🚀 核心功能

📚

智能單字卡

根據您的程度（國小至大學）和興趣類別，自動產生專業單字卡。包含音標、詞性、定義、例句、同義詞、反義詞，還能一鍵聽發音！

📝

句型學習卡

涵蓋疑問句、條件句、被動語態等 12 種句型類型，配合真實情境例句，讓您快速掌握英文句法結構。

🎤

情境口說練習

模擬餐廳、機場等真實情境，支援語音輸入與 AI 對話練習。系統會即時糾正發音和文法錯誤，幫助您自信開口說英文。

📊

智能測驗系統

透過 Dify AI 技術，提供個性化測驗題目，即時評估學習成果，讓您了解自己的學習進度。

🔊

語音互動

支援語音轉文字（Speech-to-Text）和文字轉語音（TTS），讓學習過程更自然、更便利。

📸

OCR 文字識別

上傳圖片即可自動識別文字內容，支援多語言辨識，讓您隨時隨地學習新單字。

💡 特色亮點

  • 狀態管理系統：記住您的學習偏好與進度
  • 模組化設計：可隨時切換不同學習模式
  • 個人化體驗：根據程度自動調整內容難度
  • 即時回饋：AI 即時糾正錯誤並提供建議
  • 多樣化學習：單字、句型、口說、測驗一次滿足

🛠️ 技術架構

採用現代化技術棧，確保穩定、高效、可擴展的服務體驗

LINE Bot SDK OpenAI GPT-4o OpenAI TTS Speech-to-Text Firebase Functions Firebase Storage Supabase Dify API Express.js Node.js

🔧 技術特點

  • ✅ 雲端架構：基於 Firebase Functions 的 Serverless 架構，自動擴展
  • ✅ AI 整合：深度整合 OpenAI 多種模型（GPT-4o、TTS、Whisper）
  • ✅ 狀態管理：智能記憶使用者的學習狀態與偏好設定
  • ✅ 模組化路由：使用 Express Router 實現功能模組化，易於維護
  • ✅ 即時處理：支援語音、圖片、文字等多種訊息類型的即時處理

📋 使用流程

1

添加機器人

掃描 QR Code 或點擊下方按鈕，將 LinguaBot 加入好友

2

選擇功能

輸入指令切換模式：
/切換單字卡
/切換句型卡
/切換口說
/切換測驗

3

設定偏好

選擇適合的難度等級（國小/國中/高中/大學）和學習類別

4

開始學習

點擊「產生單字/句型」或「開始對話」，立即開始您的學習之旅！

🚀 立即體驗 LanguageBot

點擊按鈕加入好友，開始您的 AI 語言學習之旅！
